# Security Policy

## Supported Versions

Security fixes are provided for the latest released version of the package.

## Reporting a Vulnerability

Please do not open a public issue for security-sensitive reports.

Send details to `manhducdongyb@gmail.com` with:

- A description of the issue.
- Steps to reproduce, if available.
- The affected package version or commit.
- Any suggested mitigation.

Maintainers will review the report and follow up with next steps.
